# ZJMediaPlayer

**商业授权请联系**：lizhijian_21@163.com

- 支持常见的H.264、HEVC、AAC、PCM等格式播放；
- 支持软硬件解码；

- 支持本地视频播放；
- 支持网络常用视频协议播放；
- 支持视频录制及视频截屏；
- 支持视频纯裸流播放；
- 支持图片合成视频；

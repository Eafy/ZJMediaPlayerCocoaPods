- 支持iOS和Android平台，具备基本一致的功能和接口；
- 支持RTMP、RTSP、HLS、本地视频和裸流数据播放；
- 支持常见的H.264、HEVC、AAC、PCM、SPEEX等格式播放；
- 支持软硬件解码；
- 支持网络视频流秒播；
- 支持视频录制、视频截屏，具备图片合成视频功能；
- 支持音频降噪和音频增益功能；
- 支持其他各种功能定制；
